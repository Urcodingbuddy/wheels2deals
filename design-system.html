10

Rules

The design language has edges. Stay inside them.

Do
  • Use Big Shoulders Display in UPPERCASE only at display sizes
  • Keep BRG as accent — prices, labels, focus rings, badges
  • Use Gulf Gold exclusively for CPO badges (max twice per page)
  • Always place the logo on a dark surface
  • Use surface-color elevation on dark pages, not box shadows
  • Keep border radius strict: 4px → 8px → 12px by nesting level
  • Let prices use Big Shoulders Display — they deserve presence
  • Scale spacing in multiples of 8
  • Apply ease-out to every entering element
  • Use the nav frosted glass on both light and dark pages
Don't
  • Never use BRG or Gold as a CTA background — buttons are black or white only
  • Never mix dark and light surfaces on the same page
  • Never use gradients on UI chrome or buttons
  • Never use thin or italic font weights
  • Never put the logo on the ivory background without a dark container
  • Never use custom or static px font sizes — use the type scale
  • Never animate for decoration — every motion must serve clarity
  • Never use more than 2 accent colors on a single view
  • Never nest grids inside grids
  • Never use a border radius that isn't 4, 8, 12, or 9999px
